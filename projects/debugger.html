Smart Debugger

When learning C programming (after learning Java), I found it frustrating trying to figure out and visualize the layout of my data structures, both abstractly and in memory.

This project aims to alleviate that frustration by providing a tool to visualize variables, data, and heap allocations in an easy to use debugging tool. I currently have completed multiple proof-of-concepts, and am moving towards architecting the overall system.

I've found a multitude of Rust libraries that make it significantly easier to parse debug information and analyze programs, but fitting everything together has proven difficult for me.
